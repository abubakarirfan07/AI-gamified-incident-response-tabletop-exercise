# AI-gamified-incident-response-tabletop-exercise
A web-based platform that transforms traditional tabletop cybersecurity exercises into an interactive, AI-driven experience with real-time scenario generation and adaptive injects.

Key Features:
AI-generated incident scenarios using GPT-4.
Real-time collaborative tabletop exercises.
After-Action Reports mapped to NIST, ISO 27035, CPS 234.
Role-Based Access Control with secure authentication.
Dockerized deployment for scalability.

Tech Stack:
Frontend: React.js, Tailwind CSS
Backend: Python Flask REST API (authentication, business logic, data routing)
AI Engine: GPT-4o for real-time scenario and inject generation
Database: Encrypted MongoDB storing users, scenarios, injects, decision logs, and reports
Deployment: Docker, AWS EC2
